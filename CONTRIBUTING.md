# Contributing

This repository is for documentation and issue tracking.

## Reporting Issues

Please report bugs and feature requests via [GitHub Issues](https://github.com/meta-engine/protobuf-csharp-httpclient/issues).

When reporting a bug, please include:
- Package version (`dotnet tool list -g | grep metaengine`)
- .NET SDK version (`dotnet --version`)
- Operating system
- Protobuf (`.proto`) definition (minimal reproducible example)
- Command used
- Expected behavior vs actual behavior
- Error messages and stack traces

## Feature Requests

We welcome feature requests! Please search existing issues first to avoid duplicates.

When suggesting a feature:
- Describe the use case and problem it solves
- Provide examples of how the API would look
- Explain why this would benefit other users

## Questions and Support

For general questions and support:
- Search [existing issues](https://github.com/meta-engine/protobuf-csharp-httpclient/issues)
